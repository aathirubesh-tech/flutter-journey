This is my first Flutter mobile application developed as part of my learning journey in Flutter and Android app development. The project was built over a couple of days while exploring the Flutter framework, Dart language, and basic UI design concepts.

The app demonstrates a simple counter functionality where the value increases each time the button is pressed. Through this project, I learned about Flutter widgets, state management basics using StatefulWidget, and how UI updates work in real time using hot reload.

This project also helped me understand how to set up a Flutter development environment, run apps on an Android emulator, and structure a basic Flutter project.

I am currently in the learning phase and план to build more advanced real-world applications in the future, such as expense trackers, to-do apps, and API-based projects.

Technologies used:

- Dart
- Android Studio
- Android Emulator
